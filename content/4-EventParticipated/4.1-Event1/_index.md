# Reflection 1: "FCAJ Community Day - Learning Mindset, Effective Prompting, AI-Ready Freshers, and the BMAD Method"

### General Information
**Event Name:** FCAJ Community Day - Learning Mindset, Effective Prompting, AI-Ready Freshers, and the BMAD Method <br>
**Time:** 09:00, May 09, 2026 <br>
**Location:** AWS / FCAJ Community Event <br>
**Role:** Attendee

### Event Objectives
The event aims to help students cultivate a self-learning mindset to be well-prepared as an "AI-ready fresher." At the same time, the event equips attendees with effective prompt writing skills and methods to systematically apply AI Agents into the software development process.

### List of Speakers
- **Mr. Long** - FCAJ Admin
- **Mr. Thinh** - DevOps / Cloud Engineer
- **Mr. Khang** - Solution Architect
- **Ms. Thao** - Software Developer

### Key Highlights
- Self-learning is easily interrupted when learners do not see immediate results. Therefore, to maintain long-term motivation, individuals need to establish discipline, create a suitable learning environment, and break things down into realistic goals. "If a task takes less than 2 minutes to complete, do it immediately"—this is a quote I found very insightful and it really stuck with me.
- Writing brief prompts like "do it" or "make it" without providing any context makes AI highly ineffective for various tasks. The speakers highlighted how significantly context impacts the output, how to improve it, and introduced tools that assist with prompting.
- Moving on to sharing how to truly be "Ready" and the current recruitment landscape: candidates must deeply understand 3-4 core services instead of trying to learn every service superficially. They also emphasized the perspective that AI is not a replacement but merely a tool to boost productivity.
- The event also introduced a very novel method to me called BMAD, which is a development approach leaning towards documentation using AI Agents. We can utilize AI Agents with multiple roles (like PM Agent, Development Agent, etc.) to assist, rather than relying on a single AI to handle an entire project.

### Lessons Learned
- After attending this event, I realized the need to practice discipline (applying the "under 2 minutes" rule) and focus on deeply learning core skills instead of spreading myself too thin to truly become an AI-Ready Fresher.
- Instead of simple Q&A, I will change how I interact with AI by writing prompts with clear context and applying systemic project thinking with AI Agents (the BMAD method).

### Application to the Project
- Apply this knowledge to design detailed, context-rich prompts so the AI can better assist in analysis, code review, and simulating specialized Agents to speed up project completion.
- Improve concentration and focus better on the group project.

### Event Experience
- The event provided an open, inspiring, and highly practical sharing space, helping me step outside the academic theories taught in university.
- The opportunity to interact and listen to "real-world" stories and experiences from senior experts gave me immense motivation for my upcoming development journey.
